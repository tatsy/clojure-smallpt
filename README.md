# clojure-smallpt

A Clojure implementation of 'smallpt' (small path tracing) that is originally coded with C.

## Usage

```shell
$ lein run [width] [height] [samples]
```

## Result

![result](./result.png)

(800 x 600 pixels, 512 samples per pixel, computation took 38000 secs with 1.8 GHz CPU)
